# Food Donation System

## Overview
A secure platform connecting food donors with receivers to reduce food wastage. Donors can list surplus food, and receivers can request donations efficiently.

## Technologies
- Backend: Python (Flask)
- Database: SQLite
- Frontend: HTML, CSS, JavaScript

## Project Structure
Food-Donation-System/
├── app.py
├── setup_db.py
├── food_donate.db
├── templates/
└── static/

## How to Run
1. Clone the repo:
git clone https://github.com/YourUsername/Food-Donation-System.git
2. Navigate to folder and create virtual environment:
cd Food-Donation-System
python -m venv venv
3. Activate environment and install dependencies:
venv\Scripts\activate   # Windows
source venv/bin/activate # Mac/Linux
pip install -r requirements.txt
4. Run the app:
python app.py
Open http://127.0.0.1:5000 in a browser.
